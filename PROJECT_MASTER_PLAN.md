# 👑 아뜨랑스 클론: "Real-World" 마스터 플랜 (Pro Ver.)

사용자님의 피드백과 실제 아뜨랑스 쇼핑몰 분석을 바탕으로, **실무 수준의 비즈니스 로직**이 포함된 최종 기획서입니다. 단순한 쇼핑몰이 아니라, **"매출을 극대화하는 플랫폼"**을 목표로 합니다.

---

## 1. 🚀 핵심 차별화 기능 (Attrangs Style)

### A. 🚚 총알 배송 시스템 (아뜨배송)
- **로직**: "오후 9시 이전 주문 시 내일 도착 보장"
- **구현**:
  - 상품 DB에 `shipping_deadline` (마감시간) 및 `stock_status` (재고상태) 필드 추가
  - 재고가 있는 상품만 '아뜨배송' 태그 자동 부착
  - 카운트다운 타이머 UI: "마감까지 03:21:55 남았습니다!"

### B. ⏱️ 타임 세일 (마케팅)
- **로직**: 특정 기간 동안만 가격 할인 + 재고 한정
- **구현**:
  - 관리자 페이지에서 `이벤트 기간`, `할인율`, `한정 수량` 설정
  - 메인 페이지 최상단에 슬라이드 배너로 노출
  - 실시간 재고 차감 시각화 ("현재 3개 남음!")

### C. 👗 코디 세트 구매 (객단가 상승)
- **로직**: "이 옷과 어울리는 상품" 추천 및 일괄 구매
- **구현**:
  - 상품 등록 시 `related_products` (관련 상품) 매핑
  - 상세 페이지에서 "코디 상품 전체 구매하기" 버튼 제공 (할인 혜택 적용)

---

## 2. 🛠️ 관리자 시스템 (Back-Office) - "사장님 모드"

### 📊 비즈니스 인텔리전스 (BI)
- **매출 대시보드**:
  - 시간대별/요일별 매출 추이 (피크 타임 분석)
  - **객단가(AOV)**, **재구매율**, **반품률** 자동 계산
  - "이번 달 목표 매출 달성률" 게이지 차트
- **엑셀(Excel) 연동**:
  - 주문서 일괄 다운로드 (택배사 양식 호환)
  - 송장 번호 일괄 업로드 및 배송 상태 자동 업데이트
  - 정산 내역서 생성

### 📦 재고 및 발주 관리
- **안전 재고 알림**: 재고가 `N`개 이하로 떨어지면 관리자에게 슬랙/문자 알림
- **입고 예정 관리**: 리오더 중인 상품의 입고 예정일 설정 및 고객 안내 ("12/5 입고 예정")

---

## 3. 🧠 알고리즘 & 데이터 (Data-Driven)

### 📈 스마트 랭킹 시스템
단순 판매량이 아닌, **"지금 뜨는 상품"**을 계산합니다.
```typescript
Score = (판매량 * 0.5) + (조회수 * 0.3) + (장바구니 담기 * 0.2) + (리뷰 평점 가중치)
// 최근 3일 데이터에 가중치를 더 높게 부여 (Trend 반영)
```

### 🎯 개인화 추천 (Basic)
- "고객님이 본 상품과 비슷한 스타일"
- 같은 카테고리 내에서 가격대/색상이 유사한 상품 추천

## 4. 💻 기술 스택 및 인프라 (Production)

### Frontend
- **Next.js 14 (App Router)**: SEO 및 초기 로딩 속도 최적화
- **TanStack Query**: 서버 상태 관리 및 캐싱
- **Framer Motion**: 고급스러운 마이크로 인터랙션 (좋아요 하트, 장바구니 담기 애니메이션)

### Backend
- **NestJS**: 엔터프라이즈급 모듈 아키텍처
- **Prisma + PostgreSQL**: 안정적인 관계형 데이터 처리
- **Redis**:
  - 실시간 랭킹 계산 (Sorted Set)
  - 세션 및 장바구니 캐싱 (속도 향상)
- **BullMQ**: 대용량 이메일/알림 발송 비동기 큐 처리

### Infra
- **AWS (EC2, RDS, S3, CloudFront)**: 실제 서비스 배포 환경
- **Docker**: 개발/운영 환경 일치
- **GitHub Actions**: 자동 배포 (CI/CD)
